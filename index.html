    • Airport API

    The Airport API can be used to retrieve information about airports. Information includes listing all major IATA airports or retrieving specific details about an airport.

     View API documentation

    Bag Tag API

    The API can be used to generate an IATA compliant bag tag in PDF format.

    View API documentation

    Beacon Registry API

    SITA Common Use Beacon Registry Community Service for Managing Global Airport iBeacon Deployments. The Beacon Registry is a registry of common use iBeacons for the Air Transport Industry (ATI).

     View API documentation

    Boarding Pass API

    Mobile boarding pass creation and delivery solution. The Boarding Pass API is a managed service provided by SITA allowing airlines complete flexibility in the creation and distribution of mobile boarding passes. IATA barcode V5 supported with bag id and Fast Track indicator.

     View API documentation

    Flight Infomation API

    Universal Flight Statistics and Information The FlightInfo API provides information and status for flights from airports around the world. The service includes subscription based notifications for status changes to a particular flight.

     View API documentation

    Flight Follower API

    The SITA FlightFollower API is a REST based service which provides map images of flights as they progress on their routes. Flight information can be searched by departure airport, arrival airport, airline and flight number. Images are returned in GIF format.

     View API documentation

    WaitTime API

    The SITA WaitTime API is a REST based service which provides information about the length of time that a passenger may expect to wait in a particular area, such as in airport security line, or customs queue.

    View API documentation

    Weather API

    The SITA Weather API is a REST based service which provides current and forecasted weather data for locations around the world.
